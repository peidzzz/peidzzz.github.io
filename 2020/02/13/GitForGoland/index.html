使用 Git 管理项目-Goland

2020年02月13日 267 字 大概 1 分钟 Git Golang
  1. 1. 下载安装Goland
  2. 2. 使用Git
  3. 3. 注意事项

我发现Goland也有内置的git管理工具，今天就来教大家如何使用Goland内置的Git来进行版本控制。

下载安装Goland

可以在官网进行下载，这里推荐使用Jetbrains ToolBox进行App管理，下载好后找到Goland直接点击Install安装。

使用Git

  1. 首先Shift+Alt+S打开 Version Control-GutHub

  1. 点击+添加Github账号，输入用户名和密码。

  1. 写好代码后 Alt+` 选择Create Git Repository，新建本地仓库。

  1. 新建好仓库后，右击需要版本控制的文件夹，选择Git-Add

  1. 选择好后，界面右上方点击Commit进行提交。

  1. 成功提交后会有提示

  2. 然后在Editer界面右键，选择Git-Repository-Push

  1. 点击Define remote在URL处输入仓库地址，点击OK、Push即可完成提交。

注意事项

  1. 已选中进行版本控制的文件在IDE中会变为绿色，没有进行版本控制的文件则为红色。
  2. 可以编辑文件夹下的.gitignore来控制提交时需要忽略的文件或文件夹。
